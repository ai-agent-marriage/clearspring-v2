# 清如项目用户操作手册

## 📖 产品简介

清如（ClearSpring）是一款 [待补充产品定位描述] 应用，包含三个核心端：
- **祈福者端**：发起祈福需求
- **执行者端**：接收并执行祈福任务
- **管理后台**：系统管理和数据监控

---

## 🙏 祈福者端使用指南

### 注册与登录

1. **下载应用**
   - iOS：App Store 搜索"清如"
   - Android：应用商店搜索"清如"
   - 或扫描官方二维码下载

2. **注册账号**
   - 打开应用，点击"注册"
   - 输入手机号，获取验证码
   - 设置登录密码（8-20 位，含字母和数字）
   - 阅读并同意用户协议
   - 完成注册

3. **登录**
   - 输入手机号和密码
   - 或使用验证码快捷登录
   - 支持微信/支付宝快捷登录

### 发起祈福

1. **创建祈福**
   - 点击首页"+"按钮
   - 选择祈福类型（健康/事业/感情/其他）
   - 填写祈福内容
   - 设置祈福预算（可选）
   - 上传相关图片（可选）
   - 点击"发布"

2. **祈福状态说明**
   - 🟡 待接单：已发布，等待执行者接单
   - 🟢 进行中：已有执行者接单，正在执行
   - 🔵 已完成：执行完成，待确认
   - ✅ 已确认：祈福者已确认完成
   - ⚪ 已取消：祈福被取消

3. **查看祈福列表**
   - 首页展示所有祈福
   - 可按状态筛选
   - 可按时间排序

### 与执行者沟通

1. **发送消息**
   - 进入祈福详情页
   - 点击底部"消息"按钮
   - 输入文字、图片、语音
   - 支持发送位置

2. **查看执行进度**
   - 祈福详情页查看进度条
   - 执行者会更新关键节点
   - 可在线留言评论

### 确认与评价

1. **确认完成**
   - 收到执行完成通知
   - 查看执行结果报告
   - 确认无误后点击"确认完成"
   - 如有问题可点击"申请复议"

2. **评价执行者**
   - 确认完成后进入评价页面
   - 评分：1-5 星
   - 填写文字评价（可选）
   - 可上传成果图片

### 常见问题

**Q: 如何修改已发布的祈福？**
A: 待接单状态下可编辑，进行中需与执行者协商

**Q: 祈福可以取消吗？**
A: 待接单状态可直接取消，进行中需双方协商

**Q: 如何退款？**
A: 联系在线客服，提供订单号和处理原因

## ✨ 执行者端使用指南

### 注册与认证

1. **注册账号**
   - 下载"清如执行者"应用
   - 使用手机号注册
   - 完成基础信息填写

2. **身份认证**
   - 上传身份证正反面
   - 进行人脸识别
   - 等待审核（1-3 个工作日）
   - 审核通过后可接单

3. **技能认证**（可选）
   - 上传相关资质证书
   - 填写专业技能描述
   - 通过平台考核
   - 获得技能标签

### 接单流程

1. **浏览祈福列表**
   - 首页展示可接祈福
   - 可按类型、距离、预算筛选
   - 查看祈福详情

2. **接单**
   - 点击感兴趣的祈福
   - 查看详细信息
   - 点击"我要接单"
   - 等待祈福者确认

3. **开始执行**
   - 确认接单后联系祈福者
   - 制定执行计划
   - 在应用中更新进度

### 执行任务

1. **更新进度**
   - 进入任务详情页
   - 点击"更新进度"
   - 填写当前进展
   - 上传图片/视频证明

2. **与祈福者沟通**
   - 使用内置聊天功能
   - 及时回复消息
   - 保持专业态度

3. **完成任务**
   - 执行完毕后上传成果
   - 填写执行报告
   - 提交完成申请
   - 等待祈福者确认

### 收入管理

1. **查看收入**
   - "我的" → "收入中心"
   - 查看总收入、待结算、已结算
   - 查看收入明细

2. **提现**
   - 绑定收款账户（微信/支付宝/银行卡）
   - 点击"提现"
   - 输入提现金额（最低 10 元）
   - 1-3 个工作日到账

3. **收入说明**
   - 祈福者确认后收入进入待结算
   - 7 天无争议后自动结算
   - 平台服务费：[待补充]%

### 常见问题

**Q: 如何提高接单成功率？**
A: 完善个人资料、提升评分、及时响应

**Q: 遇到恶意投诉怎么办？**
A: 保留证据，联系平台客服申诉

**Q: 可以同时接多个任务吗？**
A: 可以，但建议合理安排时间保证质量

## ❓ 常见问题 FAQ

### 账号相关

**Q: 一个手机号可以注册多个账号吗？**
A: 不可以，一个手机号只能绑定一个账号

**Q: 如何修改绑定手机号？**
A: 我的 → 设置 → 账号安全 → 修改手机号

**Q: 账号被盗怎么办？**
A: 立即联系客服冻结账号，提供身份证明后找回

### 祈福相关

**Q: 祈福发布后多久能被接单？**
A: 通常 1-24 小时内，具体取决于祈福类型和预算

**Q: 可以对执行者指定性别吗？**
A: 可以在发布时备注要求，平台会尽量匹配

**Q: 祈福内容有限制吗？**
A: 是的，禁止违法、违规、违背公序良俗的内容

### 支付相关

**Q: 支持哪些支付方式？**
A: 微信支付、支付宝、银行卡

**Q: 支付安全吗？**
A: 平台采用银行级加密，资金由第三方托管

**Q: 发票如何开具？**
A: 我的 → 发票中心 → 申请开票，3 个工作日内处理

### 客服相关

**Q: 如何联系客服？**
A: 
- 应用内：我的 → 帮助与客服 → 在线客服
- 电话：400-XXX-XXXX（9:00-21:00）
- 邮箱：support@clearspring.com

**Q: 客服响应时间是多久？**
A: 工作日 1 小时内响应，非工作日 4 小时内响应

**Q: 投诉建议如何提交？**
A: 我的 → 帮助与客服 → 投诉建议
