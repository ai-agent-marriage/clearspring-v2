## 🔧 技术实现

### 1. 认证机制
- **JWT Token**: 7 天有效期
- **中间件**: `authMiddleware` 统一鉴权
- **微信登录**: 通过 `jscode2session` 获取 OpenID

### 2. 分布式锁
- **实现方式**: 基于 MongoDB 的原子操作
- **TTL**: 10 秒自动过期
- **应用场景**: 抢单防并发

### 3. 断点续传
- **分片上传**: 支持大文件分片
- **状态追踪**: 记录已上传分片
- **自动合并**: 所有分片完成后自动合并

### 4. 数据库索引优化
- **唯一索引**: openId, orderNo, uploadId
- **复合索引**: userId+createdAt, executorId+status
- **TTL 索引**: audit_logs 90 天自动清理

## 🚀 部署说明

### 火山云服务器部署

1. **上传代码**
```bash
scp -r api/ root@101.96.192.63:/opt/clearspring-v2/
```

2. **安装依赖**
```bash
cd /opt/clearspring-v2/api
npm install --production
```

3. **配置环境变量**
```bash
cp .env.example .env
vim .env  # 编辑实际配置
```

4. **初始化数据库**
```bash
node database/init.js
```

5. **启动服务**
```bash
npm start
# 或使用 PM2
pm2 start server.js --name clearspring-api
```
