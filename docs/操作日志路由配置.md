try {

## 🛣️ 路由列表

### 1. 获取操作日志列表

**端点**: `GET /api/admin/audit-logs`

**权限**: 管理员（admin, super_admin）

**查询参数**:
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| page | number | 否 | 1 | 页码 |
| pageSize | number | 否 | 20 | 每页数量 |
| type | string | 否 | - | 日志类型筛选 |
| userId | string | 否 | - | 用户 ID 筛选 |
| startDate | string | 否 | - | 开始日期 (ISO 8601) |
| endDate | string | 否 | - | 结束日期 (ISO 8601) |
| keyword | string | 否 | - | 关键词搜索（用户名） |

**请求示例**:
```bash
curl -X GET "http://localhost:3000/api/admin/audit-logs?page=1&pageSize=20&type=admin_login" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

### 5. 清空日志

**端点**: `DELETE /api/admin/audit-logs`

**权限**: 超级管理员（需要 `all` 或 `manage_logs` 权限）

**查询参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| beforeDate | string | 否 | 清空该日期之前的日志 |

**请求示例**:
```bash
# 清空所有日志
curl -X DELETE "http://localhost:3000/api/admin/audit-logs" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"

# 清空 2026-01-01 之前的日志
curl -X DELETE "http://localhost:3000/api/admin/audit-logs?beforeDate=2026-01-01" \
  -H "Authorization: Bearer YOUR_ADMIN_TOKEN"
```

**响应示例**:
```json
{
  "code": "SUCCESS",
  "data": {
    "deletedCount": 1000
  },
  "message": "已删除 1000 条日志"
}
```

## 🔐 认证与授权

### 认证中间件

所有操作日志端点都需要通过 `authMiddleware` 认证：

```javascript
const { authMiddleware } = require('../../middleware/auth');
```

### 管理员权限中间件

```javascript
const adminMiddleware = (req, res, next) => {
  authMiddleware(req, res, (err) => {
    if (err) {
      return next(err);
    }
    
    // 检查是否为管理员
    if (!['admin', 'super_admin'].includes(req.user?.role)) {
      return next(new AppError('权限不足，需要管理员权限', 'FORBIDDEN', 403));
    }
    
    next();
  });
};
```

### 错误响应

| 状态码 | 错误码 | 说明 |
|--------|--------|------|
| 401 | UNAUTHORIZED | 未提供认证令牌 |
| 401 | INVALID_TOKEN | 令牌无效或过期 |
| 403 | FORBIDDEN | 权限不足 |
| 404 | LOG_NOT_FOUND | 日志不存在 |
