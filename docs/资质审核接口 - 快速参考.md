# 资质审核接口 - 快速参考

## 🚀 快速测试

```bash
# 1. 获取资质列表
curl -X GET "https://springs.dexoconnect.com/api/admin/qualifications?page=1&pageSize=10" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -k

# 2. 筛选待审核资质
curl -X GET "https://springs.dexoconnect.com/api/admin/qualifications?status=pending" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -k

# 3. 审核通过
curl -X PUT "https://springs.dexoconnect.com/api/admin/qualifications/{id}" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "approved"}' \
  -k

# 4. 审核驳回
curl -X PUT "https://springs.dexoconnect.com/api/admin/qualifications/{id}" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"status": "rejected", "rejectReason": "图片不清晰"}' \
  -k
```

## 📊 数据库集合

**集合名称**: `qualifications` (不是 certificates!)

```javascript
// 查询示例
db.qualifications.find({ status: 'pending' }).limit(10)

// 创建索引
db.qualifications.createIndex({ status: 1, createdAt: -1 })
db.qualifications.createIndex({ userId: 1 })
```
