**问题描述**: 大量 API 端点返回 `INTERNAL_ERROR` 而非预期的业务错误码或成功响应

**影响模块**:
- 📊 Dashboard 统计模块 (8/8 失败)
- 📦 订单管理模块 (12/12 失败)
- 📜 资质审核模块 (7/7 失败)
- 👷 执行者管理模块 (5/6 失败)
- 💰 分账配置模块 (6/6 失败)

**典型错误**:
```
Expected: "SUCCESS"
Received: "INTERNAL_ERROR"

Expected: "UNAUTHORIZED"
Received: "INTERNAL_ERROR"

Expected: "ORDER_NOT_FOUND"
Received: "INTERNAL_ERROR"
```

**可能原因**:
1. 后端服务未正确启动或配置
2. 数据库连接失败
3. 环境变量缺失
4. P0 修复引入了新的回归问题
