## 🔍 4. 失败用例详细分析

### 4.1 Dashboard 统计模块 (8 个失败)

**失败用例**: API-DASH-001 ~ API-DASH-011 (全部失败)

**错误模式**:
```
Expected: "SUCCESS"
Received: "INTERNAL_ERROR"
```

**根本原因**:
1. Dashboard 路由使用 `req.app.get('db')` 获取数据库连接
2. 聚合查询管道可能因集合为空或字段不匹配而失败
3. 缺少空数据容错处理

**受影响的 API**:
- `GET /api/admin/dashboard/overview` (4 个测试)
- `GET /api/admin/dashboard/orders-trend` (3 个测试)
- `GET /api/admin/dashboard/realtime` (1 个测试)

**建议修复**:
```javascript
// 添加容错处理
try {
  const stats = await db.collection('orders').aggregate([...]).toArray();
  res.json({ code: 'SUCCESS', data: stats });
} catch (error) {
  console.error('Dashboard query failed:', error);
  res.json({ code: 'INTERNAL_ERROR', message: error.message });
}
```

### 4.6 数据导出模块 (5 个失败)

**失败用例**: API-EXPT-001 ~ API-EXPT-007

**错误模式**:
```
Expected substring: "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet"
Received string: "application/json; charset=utf-8"
```

**根本原因**:
- 导出接口返回 JSON 而非 Excel/CSV 文件流
- 文件生成逻辑未实现或响应头设置错误

**受影响的 API**:
- `GET /api/admin/export/orders` (2 个测试)
- `GET /api/admin/export/executors` (1 个测试)
- `GET /api/admin/export/users` (1 个测试)

**建议修复**:
```javascript
// 使用 ExcelJS 或 xlsx 库生成文件
const workbook = new ExcelJS.Workbook();
// ... 填充数据
res.setHeader('Content-Type', 'application/vnd.openxmlformats-officedocument.spreadsheetml.sheet');
res.setHeader('Content-Disposition', 'attachment; filename="orders.xlsx"');
await workbook.xlsx.write(res);
```
