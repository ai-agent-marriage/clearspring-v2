```json
  "content": "Markdown 格式内容",
  "coverImage": "封面图 URL",
  "readCount": 0,
  "createdAt": "2026-03-30T00:00:00Z",
  "status": "published"
}
```

## 文件位置

- **内容文件**: `/content/obu-wiki/` (100 个 JSON 文件)

- **数据文件**: `/database/wiki-data.json` (聚合数据)
- **生成脚本**: `/content/obu-wiki/generate_articles.py`
- **本文档**: `/content/obu-wiki/CONTENT_SUMMARY.md`

## 使用方法

### 1. 运行导入脚本

```bash
cd /root/.openclaw/workspace/projects/clearspring-v2
node database/import-wiki-data.js
```

### 2. 生成的数据文件

运行脚本后生成 `wiki-data.json`，包含所有 100 篇文章的完整数据，可直接导入数据库。

### 3. 小程序端调用

小程序端可通过 API 获取百科内容：
- 按分类获取
- 按标签搜索
- 获取单篇详情
- 统计阅读次数
