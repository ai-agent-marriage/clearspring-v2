### 4. 后端 API（4 个新增接口）

**文件位置**: `/api/routes/content/`

**路由结构**:
```javascript
// api/routes/content/index.js
router.use('/admin/wiki', wikiRoutes);      // 百科管理
router.use('/meditation', meditationRoutes); // 冥想 API
router.use('/ritual', ritualRoutes);         // 仪轨 API
```

**API 接口清单**:

**百科内容 API** (`/api/content/admin/wiki/*`):
- `POST /create` - 创建百科
- `PUT /:id` - 更新百科
- `DELETE /:id` - 删除百科
- `GET /list` - 百科列表
- `GET /:id` - 百科详情
- `GET /category/:category` - 按分类获取
- `POST /import` - 批量导入

**冥想 API** (`/api/content/meditation/*`):
- `GET /courses` - 课程列表
- `GET /courses/:id` - 课程详情
- `POST /record` - 记录冥想
- `GET /stats` - 统计数据
- `GET /badges` - 徽章列表

**仪轨 API** (`/api/content/ritual/*`):
- `GET /rituals` - 仪轨列表
- `GET /rituals/:id` - 仪轨详情
- `POST /progress` - 记录进度
- `GET /progress` - 获取进度

**数据库集合** (5 个):
1. `wiki_contents` - 百科内容
2. `meditation_courses` - 冥想课程
3. `meditation_records` - 冥想记录
4. `ritual_contents` - 仪轨内容
5. `ritual_progress` - 学习进度

**索引优化**:
- 标题唯一索引
- 分类/状态/时间索引
- 用户 ID 复合索引
